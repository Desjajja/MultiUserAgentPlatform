MUAP 业务标签 Schema

版本 v1.2 | 面向 Arize Phoenix 的业务维度过滤与 AI/Platform 分层设计

设计目标：将 trace 中的 spans 明确区分为 AI 语义层 与 平台遥测层， 并为 Phoenix UI 提供可按业务维度（车道、渠道、意图、路由类型）过滤的正式标签体系。

一、分层架构

AI 语义层 模型分析、Prompt 调试、工具使用统计

Span 名称OpenInference Kind说明
router.deliver_to_agentAGENT用户一次交互的入口，包含 input/output
agent.turnCHAIN容器内一次推理周期（prompt → LLM → tool → output）
LiteLLM auto-spansLLM / TOOL / PROMPT由 LiteLLM Proxy 自动生成

统一标签：muap.layer = "ai"

平台遥测层 平台工程、容器生命周期、消息投递延迟调试

Span 名称OpenInference Kind说明
channel.cli.receiveCHAINCLI 通道消息接收
channel.feishu.receiveCHAIN飞书通道消息接收
router.routeCHAIN消息路由决策
container.wakeCHAIN容器唤醒
container.spawnCHAIN容器创建
container.killCHAIN容器终止
delivery.session.drainCHAIN会话消息批量消费
delivery.message.deliverCHAIN单条消息投递
delivery.channel.sendCHAIN通道消息发送（含 output.value）

统一标签：muap.layer = "platform"

二、业务标签注册表

2.1 核心标签（所有 span 必须携带）

标签键类型必填合法值说明
muap.layerstring必填ai / platform区分 AI 语义与平台遥测

2.2 路由标签（所有 span 建议携带）

标签键类型必填合法值说明
muap.route_typestring必填frontdesk / worker / a2a / system路由类型：前台、工作节点、A2A、系统
muap.lanestring必填frontdesk / worker业务车道
muap.channelstring必填cli / feishu / webhook入口渠道
muap.intentstring可选chat / approval / execute / system用户意图

2.3 会话标签（会话相关 span 携带）

标签键类型必填合法值说明
muap.agent_groupstring可选任意Agent Group 名称
muap.session_modestring可选任意会话模式
muap.engage_modestring可选direct / a2a交互模式

2.4 提供者标签（容器 span 携带）

标签键类型必填合法值说明
muap.providerstring可选claude / openai / sdk-openaiAI 模型提供者

四、Phoenix 过滤语法速查

4.1 UI Filter Bar（Spans 标签页）

⚠️ 大小写敏感：OpenInference span kind 必须使用大写。
span_kind == "AGENT" ✅ 有效 | span_kind == "agent" ❌ 无效
按 layer 过滤
attributes["muap.layer"].as_string() == "ai"
按路由类型过滤
attributes["muap.route_type"].as_string() == "frontdesk"
按业务车道过滤
attributes["muap.lane"].as_string() == "frontdesk"
按渠道过滤
attributes["muap.channel"].as_string() == "cli"
组合过滤（AI 语义 + 前台路由）
attributes["muap.layer"].as_string() == "ai" and attributes["muap.route_type"].as_string() == "frontdesk"
按 OpenInference kind 过滤
span_kind == "AGENT"

4.2 REST API

# 单层标签过滤
GET /v1/spans?attribute=muap.layer:ai

# 组合过滤（AND 关系）
GET /v1/spans?attribute=muap.layer:ai&attribute=muap.route_type:frontdesk

# 按 OpenInference kind 过滤
GET /v1/spans?attribute=openinference.span.kind:AGENT

4.3 Python SDK

from phoenix import Client

client = Client()
spans = client.spans.get_spans(
    project_identifier="default",
    attributes={
        "muap.layer": "ai",
        "muap.route_type": "frontdesk",
    },
)
