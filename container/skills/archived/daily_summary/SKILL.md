# 每日实验总结技能

## 运行流程

### 1. 自动汇总当日工作
从以下来源提取当日工作内容：
- `memory/YYYY-MM-DD.md`（如有）
- 当天对话上下文
- 实验记录文件

### 2. 生成结构化总结

**模板**：
```markdown
# 📅 {日期} 实验工作日志

## 🎯 核心任务
- 任务1
- 任务2

## 📊 情报调研
- [信源1] 热点内容摘要
- [信源2] 热点内容摘要

## 🧪 实验设计
- 路线A：xxx
- 路线B：xxx

## 🔧 技能新增
- 新增了 xxx skill

## 📤 知识库更新
- 上传了 xxx 文件

## ⚠️ 待处理
- 任务1
```

### 3. 存档路径
- 本地：`/home/chris/.openclaw/workspace/doc/DailySummary_{YYYYMMDD}.md`
- RAG：自动上传到知识库

## 使用方法
当用户说“总结今天”、“生成日报”、“实验存档”时触发。
