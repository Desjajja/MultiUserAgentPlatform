# 云台控制

通过MQTT远程控制二维云台。

## MQTT配置

- 服务器: `192.168.1.103:1883`
- 主题: `gimbal`

## 控制方式

```bash
bash scripts/control.sh <command>
```

### 方向命令

| 命令 | 功能 |
|------|------|
| `up` | 向上移动（俯仰180度） |
| `down` | 向下移动（俯仰0度） |
| `left` | 向左移动（偏航180度） |
| `right` | 向右移动（偏航0度） |
| `center` | 回到中心（90度） |
| `stop` | 停止移动 |

### 角度命令

设置特定角度（0-180度）：

| 命令 | 功能 |
|------|------|
| `pitch_数字` | 设置俯仰角度，如 `pitch_45` |
| `yaw_数字` | 设置偏航角度，如 `yaw_90` |
